8.2 full_join()

  • full_join() retains all rows from left- and right-hand data frames.

Example 8.1 Joining the survey data with managerial data, retaining all observations from both data frames. 
full_join(cons_dat, man_data, by = c("ResponseId", "ResponseId"))

Example 8.2 Changing which data frame is the left-hand df and which data frame is the right-hand df changes the order of the columns but not which observations are kept. 
full_join(man_data, cons_dat, by = c("ResponseId", "ResponseId"))

Example 8.3 Since the data frames only share one variable with a commmon name. dplyr does give us a lovely message to let us know what the data frames are being joined by. This message will be suppressed from this point forward. 
full_join(man_data, cons_dat)
## Joining, by = "ResponseId"
